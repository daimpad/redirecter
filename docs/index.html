Features
{}
Dateibasiert

Speichert alle Links in einer JSON-Datei. Keine Datenbank, kein Docker, kein Setup-Overhead.

→
HTTP 301

Permanente Weiterleitungen — sauber für SEO und Browser-Cache. Mit Referrer-Policy: no-referrer.

#
Klick-Zähler

Jeder Redirect inkrementiert atomar den Hit-Counter. Aufrufstatistiken direkt im Admin.

🔒
Passwortschutz

HTTP Basic Auth mit bcrypt-Hash. Redirects bleiben öffentlich, das Admin-Formular ist geschützt.

⚡
APCu-Cache

In-Memory-Cache mit 60 s TTL, automatisch nach jedem Schreibvorgang invalidiert.

⚙
Rate Limiting

Max. n Erstellungen/Minute/IP via atomares APCu oder Session-Fallback — konfigurierbar.

Installation
# Repository klonen git clone https://github.com/daimpad/redirecter.git /var/www/html/r cd /var/www/html/r # storage-Verzeichnis beschreibbar machen chmod 775 storage chmod 664 storage/data.json chown -R www-data:www-data storage # Passwort-Hash erzeugen und in config.php eintragen php -r "echo password_hash('deinPasswort', PASSWORD_BCRYPT);"
